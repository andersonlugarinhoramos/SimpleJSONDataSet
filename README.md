# Simple-JSON-DataSet for Delphi
JSON as DataSet


```delphi
Simple.JSON.DataSet;

begin
  DataSet.JSONToDataSet(JSONStringValue);
end.
```
DataSet as JSONObject


```delphi
Simple.JSON.DataSet;

begin
  DataSet.ToJSONObject;
end.
```
DataSet as JSONArray


```delphi
Simple.JSON.DataSet;

begin
  DataSet.ToJSONArray;
end.
```
